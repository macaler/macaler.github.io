Notable Projects on Github

Check out my Github! https://github.com/macaler
Every time I finish a project, I try to get it up on Github as soon as possible. Many of these projects are Codecademy Pro guided projects which I have completed as I work toward completing the Data Science path. Some are directly inspired by skills I learned through Codecademy that I wanted to get additional practice with; others result from seeing what I could do with datasets downloaded from Kaggle.

Feel free to browse my Github repositories to see what all I have done! But here are some highlighted projects that I think show off a variety of my current skills.

  • games_of_chance
    • This was a Codecademy Pro challenge project to write a few different functions that simulate games of chance. It showcases what I can do with basic Python.

  • musclehub_project
    • This was a Codecademy project designed to analyze a fictional A/B test and membership funnels. Included in this project is a PowerPoint presentation detailing my project findings as well as the project code.

  • biodiversity_project
    • This was a Codecademy project designed to practice hypothesis testing and determining the sample size needed to conduct an experiment in the context of fictional data inspired by that of U.S. National Parks. This project features a written report of my project findings as well as the project code.

  • Pokemon
    • This is an independent project I completed after downloading a dataset from Kaggle. In it, I showcase my skills with Pandas and Matplotlib, and do some hypothesis testing with SciPy.
